# Portfólio Pessoal

Site de portfólio pessoal desenvolvido como atividade prática da disciplina Aplicações web e móveis (UNINTER).

## Tecnologias utilizadas

- HTML5;
- CSS3;
- JavaScript.

## Estrutura do projeto
portfolio/

├── index.html

├── sobre.html

├── formacao.html

├── portfolio.html

├── contato.html

├── css/

│   └── style.css

├── js/

│   └── script.js

└── img/

## Páginas

- Sobre mim: apresentação pessoal, informações e hobbies;
- Formacao: formação educacional, cursos e idiomas;
- Portfolio: projetos desenvolvidos com links externos;
- Contato: formulário com validação em JavaScript.

## Funcionalidades

- Navegação entre páginas com menu fixo no topo;
- Alternância entre tema claro e escuro;
- Menu responsivo para dispositivos móveis;
- Validação de formulário em JavaScript com simulação de envio;
- Modal de confirmação apos envio do formulário.
